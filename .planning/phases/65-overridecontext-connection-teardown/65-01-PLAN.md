## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Rust ↔ DuckDB C-API | `duckdb_connect`/`duckdb_disconnect` calls cross the unsafe FFI boundary; ConnGuard's Drop is the only place `duckdb_disconnect` is invoked from new code in this phase |
